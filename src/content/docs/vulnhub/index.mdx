VulnHub hosts downloadable vulnerable virtual machines you run entirely offline. I spin these up in my home lab to practice full attack chains without touching anything I shouldn't.

These writeups cover boot2root challenges — getting from a fresh VM boot all the way to root, documenting the enumeration, exploitation, and escalation along the way.
