> Goal: send the current level's password to the service on localhost port 30000 to receive the next password.

## Approach

Connect to the port and send the current password. The service replies with the next one.

```bash frame="code" title="bandit14@bandit"
bandit14@bandit:~$ nc localhost 30000
<password>
Correct!
The password you are looking for is: <password>
```

## Why it works

The level gates the next secret behind proof you already hold the current one. The service speaks a plain line protocol over a raw TCP socket, so a bare `nc` (or `telnet`) is enough to talk to it. No encoding, no TLS.
